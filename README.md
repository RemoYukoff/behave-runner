# Behave Runner

Run Behave scenarios from CodeLens in `.feature` files.

## Usage
- Open a `.feature` file.
- Click **Run Scenario** above a `Scenario` or `Scenario Outline`.
- Output appears in both the Output Channel and an integrated Terminal.

## Notes
- The extension runs `behave` from your workspace environment.
- Scenario Outline runs execute the outline name (all examples) via `-n`.
